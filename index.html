API de Consulta CNPJ

Esta API permite consultar dados de empresas por CNPJ ou CNAE na base de dados da Receita Federal.

Endpoints

GET /api/search?tipo=cnpj&valor={cnpj}

Consulta dados de uma empresa pelo CNPJ.

Parâmetros de consulta:

Nome Tipo Descrição
tipo string Tipo de busca ('cnpj' ou 'cnae')
valor string CNPJ ou CNAE a ser consultado

GET /api/cnpj/{cnpj}

Consulta direta de dados de uma empresa pelo CNPJ.

Parâmetros de caminho:

Nome Tipo Descrição
cnpj string CNPJ da empresa (com ou sem formatação)

GET /api/cnae/{cnae}

Consulta empresas por CNAE.

Parâmetros de caminho:

Nome Tipo Descrição
cnae string Código CNAE (com ou sem formatação)

Nota: A busca por CNAE é uma funcionalidade simulada, pois a API da Receita Federal não suporta busca direta por CNAE.

Código Descrição

500 Erro interno do servidor
